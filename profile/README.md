## 🌟 About Us

JMI-OPENATOM是一个面向工程实践与开源协作的校园技术组织。

我们专注构建：

- 🧠 AI 驱动知识系统（RAG / 向量检索）
- 🏗️ 社团数字化基础设施（系统化平台）
- 🌐 全栈工程能力（Web / Desktop / Cloud）
- ⚙️ DevOps & 自动化发布体系
- 📦 校园级可复用开源项目生态

---

## 🧩 Core Projects

### 🧠 openatom-system
**社团核心管理系统 · Campus OS**

> 一体化社团数字中台（成员 / 活动 / 权限 / 数据）

- 👥 成员管理 & 组织架构
- 📅 活动发布 & 报名系统
- 🧾 表单系统 & 数据采集
- 🎯 积分 / 考核 / 激励体系
- 🔐 OAuth2 统一认证体系
- 📊 管理后台 + 数据大屏

---

### 🚀 openatom-publishSite
**校园级网站发布平台 · Mini Vercel**

> 面向学生的静态/前端项目云发布系统

- 🌍 支持 Vue / React / HTML 一键部署
- 🔗 自定义域名（CNAME 绑定）
- ⚡ 秒级发布 / 回滚机制
- 📦 多项目隔离环境
- 🔐 OAuth 登录体系
- 🧪 在线预览 & 版本管理

---

### 🧠 openatom-knowledge-os
**AI 知识库系统 · Campus Knowledge OS**

> 社团的“第二大脑”

- 📚 支持 Word / PPT / Excel / Markdown
- 🔍 全文 + 向量语义检索
- 🤖 AI 对话式知识访问（RAG）
- 🧩 标签 / 分类 / 知识图谱
- 🔗 与业务系统深度联动
- 🧠 自动沉淀社团知识资产
